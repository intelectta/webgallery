About this Project

WEBGALLERY is a local-to-web generator built in Python + Tkinter. It scans selected media files (MP3/MP4/WAV) and creates interactive HTML pages with configurable fields (Title, Duration, Path) and action buttons (Play, Download, Like).

How to Add Your Own Gallery

  1. Run WEBGALLERY.py on your computer.
  2. Generate a new gallery (HTML file).
  3. Move the generated file into this /docs/ folder.
  4. Commit and push to GitHub:
git add docs/gallery_myshow.html
git commit -m "Add new gallery"
git push

Then it will appear in the list above.
